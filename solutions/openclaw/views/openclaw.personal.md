## OpenClaw（个人版）

### 核心

让AI不仅回答问题，还能直接帮你执行任务。

---

### 快速使用

安装：

```bash
npm install -g openclaw
```

启动：

```bash
openclaw gateway --port 18789
```

访问：
http://localhost:18789

---

### 可以做什么

- 查询信息
- 自动执行简单任务
- 调用API获取数据
