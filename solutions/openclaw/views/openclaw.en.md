## 0. Core Value

> OpenClaw = Transform AI from a "chat tool" into an "execution system"

Core capability:

> Understand → Decide → Execute

- Understand user intent  
- Decide execution path  
- Execute via system integration  

This capability is consistent across architecture, execution flow, and use cases.

---

## 1. Background

Traditional enterprise systems store data, while humans execute tasks. AI today mostly answers questions but cannot execute business actions.

> AI understands, but cannot execute.

OpenClaw bridges this gap by connecting models to enterprise systems.

---

## 2. Architecture

User → Gateway → Agent → Model → Tools → Enterprise Systems → Response

This transforms natural language into real actions.

---

## 3. Deployment

Install:

```bash
npm install -g openclaw
```

Start:

```bash
openclaw gateway --port 18789
```

Access:
http://localhost:18789
